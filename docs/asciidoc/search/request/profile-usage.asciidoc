[[profile-usage]]
== Profile Usage

=== Fluent DSL Example

[source,csharp]
----
s => s
.Profile()
.Query(q => q
    .Term(p => p.Name, "elasticsearch")
)
----

=== Object Initializer Syntax Example

[source,csharp]
----
new SearchRequest<Project>
{
    Profile = true,
    Query = new TermQuery
    {
        Field = "name",
        Value = "elasticsearch"
    }
}
----

[source,javascript]
.Example json output
----
{
  "profile": true,
  "query": {
    "term": {
      "name": {
        "value": "elasticsearch"
      }
    }
  }
}
----
