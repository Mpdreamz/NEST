=== Fluent DSL Example

[source,csharp,method="fluent"]
----
s => s
.IndicesBoost(b => b
	.Add("index1", 1.4)
	.Add("index2", 1.3)
)
----

=== Object Initializer Syntax Example

[source,csharp,method="initializer"]
----
new SearchRequest<Project>
{
	IndicesBoost = new Dictionary<IndexName, double>
	{
			{ "index1", 1.4 },
			{ "index2", 1.3 }
	}
}
----

.Example json output
----

  "indices_boost": {
    "index1": 1.4,
    "index2": 1.3
  }
}
----
