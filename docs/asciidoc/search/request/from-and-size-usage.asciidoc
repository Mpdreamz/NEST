== From And Size Usage

=== Object Initializer Syntax Example

[source,csharp]
----
new SearchRequest<Project>()
{
    From = 10,
    Size = 12
}
----

[source,javascript]
.Example json output
----
{
  "from": 10,
  "size": 12
}
----

=== Fluent DSL Example

[source,csharp]
----
s => s
    .From(10)
    .Size(12)
----
