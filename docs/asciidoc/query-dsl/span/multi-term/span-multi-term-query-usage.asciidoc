== Span Multi Term Query Usage

[source,javascript]
.Example json output
----
{
  "span_multi": {
    "_name": "named_query",
    "boost": 1.1,
    "match": {
      "prefix": {
        "name": {
          "value": "pre-*"
        }
      }
    }
  }
}
----

=== Object Initializer Syntax Example

[source,csharp]
----
new SpanMultiTermQuery
		{
Name = "named_query",
Boost = 1.1,
Match = new PrefixQuery { Field = "name", Value = "pre-*" }
		}
----
