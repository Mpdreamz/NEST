== Span First Query Usage

[source,javascript]
.Example json output
----
{
  "span_first": {
    "_name": "named_query",
    "boost": 1.1,
    "match": {
      "span_term": {
        "name": {
          "value": "value"
        }
      }
    },
    "end": 3
  }
}
----

=== Object Initializer Syntax Example

[source,csharp]
----
new SpanFirstQuery
		{
Name = "named_query",
Boost = 1.1,
End = 3,
Match = new SpanQuery
{
	SpanTerm = new SpanTermQuery { Field = "name", Value = "value" }
}
		}
----
