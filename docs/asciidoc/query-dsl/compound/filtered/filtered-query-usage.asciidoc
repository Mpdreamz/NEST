== Filtered Query Usage

[source,javascript]
.Example json output
----
{
  "filtered": {
    "_name": "named_query",
    "boost": 1.1,
    "filter": {
      "match_all": {
        "_name": "filter"
      }
    },
    "query": {
      "match_all": {
        "_name": "query"
      }
    }
  }
}
----

=== Object Initializer Syntax Example

[source,csharp]
----
new FilteredQuery()
		{
Name = "named_query",
Boost = 1.1,
Filter = new MatchAllQuery { Name ="filter" },
Query = new MatchAllQuery() { Name = "query" },
		}
----
