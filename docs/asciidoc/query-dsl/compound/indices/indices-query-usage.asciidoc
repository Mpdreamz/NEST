== Indices Query Usage

[source,javascript]
.Example json output
----
{
  "indices": {
    "_name": "named_query",
    "boost": 1.1,
    "indices": [
      "project"
    ],
    "no_match_query": {
      "match_all": {
        "_name": "no_match"
      }
    },
    "query": {
      "match_all": {}
    }
  }
}
----

=== Object Initializer Syntax Example

[source,csharp]
----
new IndicesQuery()
		{
Name = "named_query",
Boost = 1.1,
Indices = Index<Project>(),
Query = new MatchAllQuery(),
NoMatchQuery = new MatchAllQuery { Name ="no_match" }

		}
----
