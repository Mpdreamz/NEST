== Nested Query Usage

[source,javascript]
.Example json output
----
{
  "nested": {
    "_name": "named_query",
    "boost": 1.1,
    "query": {
      "match_all": {}
    },
    "path": "curatedTags",
    "inner_hits": {
      "explain": true
    }
  }
}
----

=== Object Initializer Syntax Example

[source,csharp]
----
new NestedQuery
		{
Name = "named_query",
Boost = 1.1,
InnerHits = new InnerHits { Explain = true },
Query = new MatchAllQuery(),
Path = Field<Project>(p=>p.CuratedTags)
		}
----
