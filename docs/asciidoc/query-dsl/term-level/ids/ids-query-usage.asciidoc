== Ids Query Usage

[source,javascript]
.Example json output
----
{
  "ids": {
    "_name": "named_query",
    "boost": 1.1,
    "types": [
      "project",
      "developer"
    ],
    "values": [
      1,
      2,
      3,
      4
    ]
  }
}
----

=== Object Initializer Syntax Example

[source,csharp]
----
new IdsQuery
		{
Name = "named_query",
Boost = 1.1,
Values = new List<Id> { 1, 2,3,4 },
Types = Type<Project>().And<Developer>()
		}
----
