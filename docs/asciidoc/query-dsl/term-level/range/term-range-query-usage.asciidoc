== Term Range Query Usage

[source,javascript]
.Example json output
----
{
  "range": {
    "description": {
      "_name": "named_query",
      "boost": 1.1,
      "gt": "foo",
      "gte": "foof",
      "lt": "bar",
      "lte": "barb"
    }
  }
}
----

=== Object Initializer Syntax Example

[source,csharp]
----
new TermRangeQuery
		{
Name = "named_query",
Boost = 1.1,
Field = "description",
GreaterThan = "foo",
GreaterThanOrEqualTo = "foof",
LessThan = "bar",
LessThanOrEqualTo = "barb"
		}
----
