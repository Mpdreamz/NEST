== Term Query Usage

[source,javascript,method="queryjson"]
.Example json output
----
{
  "term": {
    "description": {
      "_name": "named_query",
      "boost": 1.1,
      "value": "project description"
    }
  }
}
----

=== Object Initializer Syntax Example

[source,csharp,method="queryinitializer"]
----
new TermQuery
		{
Name = "named_query",
Boost = 1.1,
Field = "description",
Value = "project description"
		}
----
