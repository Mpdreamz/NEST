[[prefix-query-usage]]
== Prefix Query Usage

[source,javascript]
.Example json output
----
{
  "prefix": {
    "description": {
      "_name": "named_query",
      "boost": 1.1,
      "rewrite": "top_terms_boost_N",
      "value": "proj"
    }
  }
}
----

=== Object Initializer Syntax Example

[source,csharp]
----
new PrefixQuery
		{
Name = "named_query",
Boost = 1.1,
Field = "description",
Value = "proj",
Rewrite = RewriteMultiTerm.TopTermsBoostN
		}
----
