== Terms Lookup Query Usage

[source,javascript]
.Example json output
----
{
  "terms": {
    "_name": "named_query",
    "boost": 1.1,
    "description": {
      "id": 12,
      "index": "devs",
      "path": "lastName",
      "type": "developer"
    }
  }
}
----

=== Object Initializer Syntax Example

[source,csharp]
----
new TermsQuery
		{
Name = "named_query",
Boost = 1.1,
Field = "description",
TermsLookup = new FieldLookup
{
	Id = 12,
	Index = Index<Developer>(),
	Type = Type<Developer>(),
	Path = Field<Developer>(p=>p.LastName)
}
		}
----
