=== Fluent DSL Example

[source,csharp,method-name="fluent"]
----
s => s
.Aggregations(a => a
	.Missing("projects_without_a_description", m => m
		.Field(p => p.Description)
	)
)
----

=== Object Initializer Syntax Example

[source,csharp,method-name="initializer"]
----
new SearchRequest<Project>
{
	Aggregations = new MissingAggregation("projects_without_a_description")
	{
		Field = Field<Project>(p => p.Description)
	}
}
----

=== Handling Responses

[source,csharp,method-name="expectresponse"]
