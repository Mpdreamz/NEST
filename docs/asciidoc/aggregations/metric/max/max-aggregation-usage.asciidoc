=== Fluent DSL Example

[source,csharp,method-name="fluent"]
----
s => s
.Aggregations(a => a
	.Max("max_commits", m => m
		.Field(p => p.NumberOfCommits)
	)
)
----

=== Object Initializer Syntax Example

[source,csharp,method-name="initializer"]
----
new SearchRequest<Project>
{
	Aggregations = new MaxAggregation("max_commits", Field<Project>(p => p.NumberOfCommits))
}
----

=== Handling Responses

[source,csharp,method-name="expectresponse"]
----
response.IsValid.Should().BeTrue();
var max = response.Aggs.Max("max_commits");
max.Should().NotBeNull();
max.Value.Should().BeGreaterThan(0);
----
